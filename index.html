Project Title: Smart Mirror – Interactive IoT Display Powered by Raspberry Pi

An award-winning electronics hardware project that secured First Runner-Up at the KEC LITE Technical Exhibition (Kantipur Engineering College).

Technologies Used:
• Hardware: Raspberry Pi (microcontroller), temperature & pressure sensors, mirror/display module
• Software: JavaScript (application logic), Node.js/Express (backend), HTML/CSS (UI)
• Connectivity: Wi-Fi for IoT connectivity and remote updates

What the Project Serves:
• Real-time clock and live weather data (temperature, pressure) with a 7-day forecast
• Integrated to-do list and personalized greetings (e.g., “Good Morning”, “You’re Looking Good”)
• Remote customization of displayed content via a dedicated web application
• Demonstrates how embedded computing and web technologies enhance everyday objects

Project Title: Color Sorting System — Automated Sorting Using Arduino:

Color Sorting System — Automated Sorting Using Arduino:

An embedded hardware project designed to automatically identify and sort objects based on their color. This system was built using an Arduino microcontroller, a TCS3200 color sensor module, and servo motors to physically move items into their respective bins.

Technologies Used:
• Arduino (microcontroller and control logic)
• TCS3200 color sensor module for detecting object colors
• Servo motors for mechanical sorting action
• Supporting components for power and structure

Purpose / What the Project Serves:
• Automatically detects the color of an object placed on the sensor
• Rotates or actuates servo motors to direct the object into the correct bin based on its color
• Demonstrates the integration of sensors and actuators with Arduino for real-time decision making
• Provides a low-cost, scalable approach to automated color-based sorting useful in small-scale manufacturing, packaging, or educational demonstrations
